This is a Python web server running with Flask on a Raspberry Pi Linux computer.
The web server provides a REST interface for easy interaction.
See blog post.
  • Server Status:
    /status
    /params
  • Record:
    /record/1
    /record/0
  • Stream:
    /stream/1
    /stream/0
  • Lights:
    /irLED/1
    /irLED/0
    /whiteLED/1
    /whiteLED/0
  • Images
    /lastimage
  • Set
    /set/fps/<int:value>
    /set/fileDuration/<int:value>
